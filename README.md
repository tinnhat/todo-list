- HTML/CSS(SCSS)
## Feature:
+ Create new task & description
+ Edit task & description
+ Update task to complete
+ Delete task
+ Support Responsive
## Usage
Run in local.
```bash
npm install
npm run dev
```
> [!NOTE]  
> This project using node version 20.10.0, Please use the same version to avoid errors.
